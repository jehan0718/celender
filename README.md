# 📅 상담 스케줄 관리 시스템 (Counseling Schedule System)

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-ISC-green)

팀원들이 함께 사용할 수 있는 실시간 공유 상담 스케줄 관리 서비스입니다. 본 문서는 배포된 버전의 사용 방법과 서버 설정에 대해 안내합니다.

---

## ✨ 주요 기능

- **📅 주간 캘린더 뷰**: 2026년 날짜를 기반으로 한 직관적인 주간 스케줄 확인.
- **👥 다중 상담사 관리**: 여러 상담사의 일정을 한눈에 파악하고 필터링할 수 있습니다.
- **🎨 직관적인 UI**: 상담사별로 지정된 색상을 통해 일정을 쉽게 구분합니다.
- **🔄 실시간 데이터 동기화**: 모든 팀원이 동일한 데이터를 실시간으로 공유하고 업데이트합니다.
- **📱 반응형 웹**: PC와 태블릿 등 다양한 기기에서 최적화된 화면을 제공합니다.

---

## 🚀 배포 및 서버 실행 방법

이 프로젝트는 Node.js 기반의 Express 서버를 사용합니다.

### 1. 서비스 웹사이트 접속 (배포된 경우)
관리자가 제공한 URL(예: `https://your-counseling-app.render.com`)로 접속하면 바로 사용이 가능합니다. 별도의 설치가 필요하지 않습니다.

### 2. 직접 배포하기 (관리자용)
클라우드 서비스(Render, Heroku, Vercel 등)를 통해 배포할 수 있습니다.

**주요 환경 설정:**
- **Runtime**: Node.js
- **Build Command**: `npm install`
- **Start Command**: `node server.js`
- **Port**: 기본값 `3000` (환경 변수 `PORT` 지원)

## 📖 사용 가이드

### 1. 스케줄 추가
- **방법 A**: 상단의 **[스케줄 추가]** 버튼을 클릭합니다.
- **방법 B**: 달력의 비어있는 **시간대 셀**을 직접 클릭합니다.
- **입력 항목**: 상담사 선택, 날짜, 시간, 내담자명, 회기 차수(예: 1회기)를 입력하고 저장합니다.

### 2. 스케줄 상세 확인 및 수정
- 등록된 스케줄 카드를 클릭하면 상세 정보를 확인하거나 내용을 수정할 수 있습니다.

### 3. 스케줄 삭제
- 스케줄 카드 위에서 **마우스 오른쪽 버튼(우클릭)**을 누르면 삭제 확인 팝업이 나타납니다.

### 4. 상담사별 필터링
- 상단의 상담사 선택 드롭다운 메뉴를 통해 특정 상담사의 일정만 골라볼 수 있습니다.

## 💾 데이터 관리 및 보안

- **데이터 저장**: 모든 스케줄 정보는 서버의 `schedules.json` 파일에 안전하게 기록됩니다.
- **백업**: 정기적으로 서버 내의 `schedules.json` 파일을 백업하는 것을 권장합니다.
- **접근 권한**: 본 시스템은 간편한 사용을 위해 별도의 로그인 과정이 없습니다. 따라서 URL 주소가 외부로 유출되지 않도록 주의해 주세요.
