# 상담 스케줄 관리 애플리케이션

2026년 캘린더 기반 상담팀 공유 스케줄 관리 웹 애플리케이션

## 기능

- 📅 2026년 월별/주별 캘린더 네비게이션
- ⏰ 08:00~21:00 시간대 스케줄 관리
- 👥 상담사별 색상 구분 및 필터링
- 📝 내담자명, 상담시간, 회기 정보 관리
- 💾 로컬 스토리지 자동 저장
- 🎨 다크 모드 디자인

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 월/주 선택
3. 원하는 시간대 클릭하여 스케줄 추가
4. 스케줄 카드 클릭으로 수정
5. 우클릭으로 삭제

## 기술 스택

- HTML5
- CSS3 (다크 모드, 그라디언트, 애니메이션)
- Vanilla JavaScript
- Local Storage API

## 파일 구조

```
├── index.html    # 메인 HTML
├── styles.css    # 스타일시트
└── app.js        # JavaScript 로직
```
