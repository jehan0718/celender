# 상담 스케줄 관리 시스템

팀원들이 함께 사용할 수 있는 공유 상담 스케줄 관리 애플리케이션입니다.

## 기능

- 📅 2026년 주간 캘린더 뷰
- 👥 여러 상담사의 스케줄 관리
- 📝 내담자명, 시간, 회기 정보 입력
- 🔄 실시간 데이터 동기화 (여러 컴퓨터에서 동시 사용 가능)
- 🎨 상담사별 색상 구분
- ✏️ 스케줄 수정 및 삭제 (우클릭)

## 설치 방법

### 1. Node.js 설치 확인

먼저 Node.js가 설치되어 있는지 확인하세요:

```bash
node --version
```

Node.js가 설치되어 있지 않다면 [nodejs.org](https://nodejs.org/)에서 다운로드하여 설치하세요.

### 2. 의존성 설치

프로젝트 폴더에서 다음 명령어를 실행하세요:

```bash
npm install
```

## 실행 방법

### 서버 시작

```bash
npm start
```

서버가 시작되면 다음과 같은 메시지가 표시됩니다:
```
Server running at http://localhost:3000
API available at http://localhost:3000/api/schedules
```

### 애플리케이션 접속

웹 브라우저에서 다음 주소로 접속하세요:

```
http://localhost:3000
```

## 팀원들과 함께 사용하기

### 같은 네트워크에서 사용하는 경우

1. **서버를 실행하는 컴퓨터에서:**
   - 서버를 시작합니다 (`npm start`)
   - 자신의 IP 주소를 확인합니다:
     ```bash
     ipconfig
     ```
     (IPv4 주소를 찾으세요, 예: 192.168.0.100)

2. **다른 팀원들의 컴퓨터에서:**
   - 웹 브라우저에서 `http://[서버IP주소]:3000`으로 접속
   - 예: `http://192.168.0.100:3000`

### 각자의 컴퓨터에서 개별적으로 사용하는 경우

각 팀원이 자신의 컴퓨터에서:
1. 이 프로젝트 폴더를 복사
2. `npm install` 실행
3. `npm start`로 서버 시작
4. `http://localhost:3000`으로 접속

**주의:** 이 경우 각자의 데이터가 별도로 저장되므로, 공유하려면 같은 서버를 사용해야 합니다.

## 사용 방법

### 스케줄 추가
1. "스케줄 추가" 버튼 클릭 또는 원하는 시간대의 셀 클릭
2. 상담사명, 날짜, 시간, 내담자명, 회기 입력
3. "저장" 버튼 클릭

### 스케줄 수정
- 스케줄 항목을 클릭하여 수정

### 스케줄 삭제
- 스케줄 항목에서 우클릭 후 확인

### 필터링
- 상단의 드롭다운에서 특정 상담사의 스케줄만 보기

## 데이터 저장

모든 스케줄 데이터는 `schedules.json` 파일에 저장됩니다. 이 파일을 백업하면 데이터를 보관할 수 있습니다.

## 문제 해결

### 서버가 시작되지 않는 경우
- 3000번 포트가 이미 사용 중일 수 있습니다
- `server.js` 파일에서 `PORT` 값을 다른 번호로 변경하세요

### 데이터가 저장되지 않는 경우
- 서버가 실행 중인지 확인하세요
- 브라우저 콘솔(F12)에서 에러 메시지를 확인하세요
- `schedules.json` 파일에 쓰기 권한이 있는지 확인하세요

### 다른 컴퓨터에서 접속이 안 되는 경우
- 방화벽 설정을 확인하세요
- 같은 네트워크에 연결되어 있는지 확인하세요
- 서버 컴퓨터의 IP 주소가 정확한지 확인하세요
