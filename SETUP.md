# 상담 스케줄 관리 시스템 - 설정 가이드

## ⚠️ 중요: 먼저 Node.js를 설치해야 합니다!

### Node.js 설치 방법

1. **Node.js 다운로드**
   - [https://nodejs.org/](https://nodejs.org/) 접속
   - "LTS" (Long Term Support) 버전 다운로드 (권장)
   - Windows 설치 파일(.msi) 다운로드

2. **설치 진행**
   - 다운로드한 파일 실행
   - 설치 마법사의 안내에 따라 진행
   - 모든 기본 설정 그대로 사용 (Next 클릭)
   - "Automatically install the necessary tools" 체크박스는 선택하지 않아도 됩니다

3. **설치 확인**
   - 새로운 PowerShell 또는 명령 프롬프트 창 열기
   - 다음 명령어 입력:
   ```bash
   node --version
   npm --version
   ```
   - 버전 번호가 표시되면 설치 성공!

### 설치 후 다음 단계

Node.js 설치가 완료되면:

1. **의존성 설치**
   ```bash
   npm install
   ```

2. **서버 시작**
   ```bash
   npm start
   ```

3. **브라우저에서 접속**
   ```
   http://localhost:3000
   ```

## 변경 사항

✅ **이제 데이터가 서버에 저장됩니다!**
- localStorage 대신 서버의 `schedules.json` 파일에 저장
- 여러 컴퓨터에서 같은 서버에 접속하면 데이터가 실시간으로 동기화됩니다
- 스케줄 추가, 수정, 삭제가 모두 서버에 저장됩니다

## 팀원들과 공유하는 방법

### 옵션 1: 한 대의 컴퓨터를 서버로 사용 (권장)

1. **서버 컴퓨터 (1대):**
   - Node.js 설치
   - `npm install` 실행
   - `npm start`로 서버 시작
   - IP 주소 확인: `ipconfig` 명령어 실행 (IPv4 주소 확인)

2. **다른 팀원들:**
   - 웹 브라우저만 있으면 됩니다
   - `http://[서버IP]:3000` 접속
   - 예: `http://192.168.0.100:3000`

### 옵션 2: 각자의 컴퓨터에서 실행

- 각 팀원이 자신의 컴퓨터에서 서버를 실행
- 데이터는 각자 별도로 저장됨
- 공유가 필요하면 옵션 1 사용

## 사용 방법

### 스케줄 추가
1. "스케줄 추가" 버튼 클릭 또는 원하는 시간대 셀 클릭
2. 정보 입력 (상담사명, 날짜, 시간, 내담자명, 회기)
3. "저장" 클릭 → 서버에 자동 저장됨

### 스케줄 수정
- 스케줄 항목 클릭 → 정보 수정 → "저장"

### 스케줄 삭제
- 스케줄 항목에서 **우클릭** → 삭제 확인

### 필터링
- 상단 드롭다운에서 특정 상담사 선택

## 데이터 백업

`schedules.json` 파일을 복사하여 백업할 수 있습니다.

## 문제 해결

### "npm을 찾을 수 없습니다" 오류
→ Node.js를 설치하고 **새로운** PowerShell 창을 열어야 합니다

### 서버가 시작되지 않음
→ 3000번 포트가 사용 중일 수 있습니다. `server.js`에서 PORT를 3001로 변경하세요

### 다른 컴퓨터에서 접속 안 됨
→ 방화벽 설정 확인 및 같은 네트워크 연결 확인
