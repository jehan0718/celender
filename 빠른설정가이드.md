---

## 🚀 3단계로 완성하기

### 1️⃣ 스프레드시트 헤더 설정 (1분)

1. 위 URL로 스프레드시트 열기
2. 첫 번째 행에 다음 헤더 입력:

```
id | counselor | date | startTime | endTime | clientName | sessionNumber
```

**팁**: A1부터 G1까지 순서대로 입력하세요.

---

### 2️⃣ Apps Script 설정 (3분)

1. 스프레드시트 메뉴: **확장 프로그램 > Apps Script**
2. 기본 코드 삭제
3. 프로젝트의 `Code.gs` 파일 내용 **전체 복사 → 붙여넣기**
4. 저장 (Ctrl+S)
5. **배포 > 새 배포** 클릭
6. 유형: **웹 앱** 선택
7. 설정:
   - 실행: **나**
   - 액세스: **전체 사용자**
8. 배포 → 권한 승인
9. **웹 앱 URL 복사** ⭐ (중요!)

---

### 3️⃣ 로컬 서버 설정 (1분)

1. `.env` 파일 열기
2. 복사한 웹 앱 URL 붙여넣기:

```env
GOOGLE_SCRIPT_URL=복사한_URL_붙여넣기
```

3. 서버 시작:
```bash
npm start
```

4. 브라우저 접속:
```
http://localhost:3000
```
